## Dependencias

node version 18.14.1 e mongodb ou docker

## Instalação sem docker

```bash
$ yarn install
```

## Rodar o app sem docker

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Uso com docker ambiente DEV

```bash
$ sudo chmod +x start.sh && docker compose up
```
